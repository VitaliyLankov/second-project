У нас есть блок с тремя вкладками. Кнопки для переключения между вкладками выстроились в одну строку. Нужно сделать так, чтобы кнопки для переключения вкладок перестроились в столбец — одна под другой. Чтобы добиться такого эффекта, достаточно изменить направление основной оси — она должна смотреть не слева направо, а сверху вниз. И поможет это сделать свойство flex-direction. Когда у нас стоит wrap на куче элементов флекса, которые не помещаются в контейнер без врапа, управлять их расположением по горизонатли align-items не получится, тк это типа как одна строка все. Для этого есть align-content. Пример выше

Порядок элементов в флексе. Order 1, order 2, order 3. Задаестя в css

align-self - когда нужно одному элементу придать особый порядок. Flex-grow:1 когда нужно, чтобы элемент занял все свободно пространство

Решить эту задачу поможет свойство flex-basis. Оно указывает размер элемента до того, как на него начнёт действовать расширение. Значением flex-basis может быть размер в любых относительных или абсолютных единицах. Значение по умолчанию — auto. В таком случае элемент занимает столько пространства, сколько требуется для отображения его контента.

Значением flex-shrink может быть любое положительное целое число. Отрицательные значения игнорируются. По умолчанию его значение 1 — то есть все элементы сжимаются с одинаковой скоростью. Чем больше значение flex-shrink, тем быстрее сжимается элемент. Элементы, у которых установлено свойство flex-shrink: 0, не сжимаются. Их размер не может стать меньше, чем тот, что задан с помощью flex-basis. Значение по умолчанию — flex: 0 1 auto. Расшифровывается это так: flex-grow: 0; flex-shrink: 1; flex-basis: auto;

Позиционирование

Элемент, спозиционированный абсолютно, вырван из потока. Его пространство в потоке заняли элементы, которые следовали за ним.

Grids

блок с абсолютным позиционированием попробует найти другие родительские элементы с нестатичным позиционированием, чтобы спозиционироваться относительно их границ.

Для управления близостью элементов используется свойство z-index. Его название говорит о том, что задействовано третье измерение. Позиционирование идёт уже не по ширине (ось x) и не по высоте (ось y), а по глубине, т.е. по оси z. Значения z-index — целые числа. Чем больше число, тем ближе к нам окажется элемент. По умолчанию у всех элементов одинаковый z-index, равный 0.
